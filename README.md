## 🚀 Deployment

The website is built with static HTML/CSS/JS and can be deployed to any web hosting platform:

### Recommended Platforms:
- **Vercel** (free, excellent performance)
- **Netlify** (free tier available)
- **GitHub Pages** (free for public repos)
- **Traditional web hosting** (any provider)

### Quick Deploy Steps:
1. Upload all files to your web server
2. Add your logo file to the `images/` directory
3. Update contact information in `contact.html`
4. Test all pages and form functionality
5. Point your domain to the hosting location

## 📋 Customization Notes

### Easy Updates:
- **Colors**: Modify CSS variables in `:root` section of `styles.css`
- **Content**: Edit HTML files directly - well-structured and commented
- **Contact Form**: Currently shows alert - integrate with your backend/email service
- **Images**: Add product photos, facility images, team photos as needed

### Contact Form Integration:
The contact form currently shows a success alert. To make it functional:
1. Set up a backend service (Node.js, PHP, Python)
2. Or use a service like Formspree, Netlify Forms, or EmailJS
3. Update the form submission handler in the JavaScript
